# 🎤 小松 AI 语音助手 - 使用指南

## ✅ 小松已成功部署！

### 🤖 关于小松：

- **名字**：小松
- **性格**：阳光开朗、稳重专业、有幽默感
- **性别**：男性
- **语音**：使用 OpenAI TTS "alloy" 音色（稳重男声）
- **能力**：语音交互 + 文字交互

### 🔑 已配置的服务：

- ✅ **Agora RTC**: 实时音视频（ID: b7d66c3f555443bebfdf3691d68d9561）
- ✅ **DeepSeek API**: LLM 对话引擎（gpt-4o）
- ✅ **OpenAI TTS**: 语音合成（alloy 男声）
- ✅ **Deepgram ASR**: 语音识别（中文）
- ✅ **Worker**: 2.4MB 完整编译版本

### 📱 访问地址：

- **Playground**: http://localhost:3000
- **TMAN Designer**: http://localhost:49483

### 🎯 使用步骤：

1. **打开浏览器**
   - 访问 http://localhost:3000
   - 或使用已打开的无痕窗口

2. **选择小松**
   - 在智能体下拉菜单中选择 **"xiaosong_voice_assistant"**

3. **连接**
   - 点击右上角的 **"Connect"** 按钮
   - 允许麦克风和摄像头权限（如需语音交互）

4. **开始对话**
   - 🎤 **语音模式**：点击麦克风按钮，直接说话
   - 💬 **文字模式**：在输入框输入文字
   - 小松会用语音和文字回复你！

### 🗣️ 小松的特点：

**欢迎语**：
> "嘿！我是小松，很高兴认识你！有什么我能帮到你的吗？"

**性格特点**：
- 说话简洁有力
- 喜欢用生动比喻
- 稳重但有亲和力
- 阳光开朗有幽默感

### 🔧 快速重启：

如果小松没反应，在 PowerShell 中运行：
```powershell
docker exec ten_agent_dev bash /app/quick-start.sh
```

然后刷新浏览器即可。

### 📊 检查服务状态：

```bash
# 检查运行中的服务
docker exec ten_agent_dev ps aux | grep -E 'api|tman|worker'

# 检查可用的智能体
curl http://localhost:8080/graphs

# 检查活跃的会话
curl http://localhost:8080/list
```

### ⚙️ 配置文件位置：

- **配置目录**: `/app/agents/examples/xiaosong/`
- **Property**: `/app/agents/examples/xiaosong/property.json`
- **Manifest**: `/app/agents/examples/xiaosong/manifest.json`

### 🎨 自定义小松：

你可以修改 `property.json` 中的：
- `prompt`: 改变小松的性格和说话方式
- `greeting`: 改变欢迎语
- `voice`: 改变语音音色（OpenAI TTS 支持多种）
- `model`: 改变 LLM 模型
